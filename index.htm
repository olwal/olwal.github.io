Covid 19 plots
[Github repository]   [Colab]

Data analysis and interactive visualizations related to the COVID-19 pandemic.

Uses data from Data Packaged Core Datasets which maintains a population dataset and a COVID-19 dataset, which is a cleaned and normalized version of the data from John Hopkins CSSE.

(Previously, data was loaded from John Hopkins CSSE with data preprocessing by Pascal Getreuer).

The plots are interactive:
  • Highlight country: Click the country name in legend
  • Highlight multiple countries: Hold down shift, while clicking additional countries in the legend
  • Zoom: Scroll to zoom in/out
  • Pan: Click + drag
  • Reset view: Double-click
